## 四、翻牌面场景测试

### 测试 14: 15% vs 30% — 干燥大牌面 `Ah Kd 7c`

```
Player 1 (15%): 56.27%   Player 2 (30%): 43.73%
```

15% 范围在 AK7 彩虹面上优势轻微扩大（翻前 55.8% → 翻后 56.3%），因为紧范围包含更多 Ax, Kx 大牌组合。

### 测试 15: 15% vs 30% — 湿润连牌面 `7h 8h 9h`

```
Player 1 (15%): 50.43%   Player 2 (30%): 49.57%
```

翻前 55.8% 的优势几乎消失！湿润牌面极大地抹平了范围差距。30% 的宽范围包含更多连牌（87s, J8s, T8s）和同花听牌，在这种牌面上缩小了差距。

### 测试 16: 5% vs 50% — 干燥中低牌面 `2c 5d Ts`

```
Player 1 (5%):  70.70%   Player 2 (50%): 29.30%
```

5% 范围（TT+ 为主）在 T52 牌面上优势进一步扩大（翻前 69.4% → 翻后 70.7%），因为 TT 直接中了暗三。

## 五、分析总结

### 1. 范围越紧，优势越大，但边际递减

```
范围比   │ 差距
─────────┼──────
5% vs 10%  │ 19.5pp
5% vs 20%  │ 29.0pp  (+9.5)
5% vs 50%  │ 38.7pp  (+9.7)
```

随着对手范围变宽，紧范围的优势持续增长，但增速逐渐放缓。

### 2. 相邻范围差距有限

```
10% vs 20%  → 15.3pp
15% vs 30%  → 11.5pp
20% vs 30%  → 6.6pp
30% vs 50%  → 10.4pp
```

**20% vs 30% 仅差 6.6 个百分点** — 这意味着在实战中，位置优势、翻后技术等因素可以轻松弥补这个范围差距。

### 3. 牌面纹理极大影响胜率

- **干燥大牌面** (AK7r): 紧范围维持甚至扩大优势
- **湿润连牌面** (789hh): 宽范围大幅缩小差距，几乎持平
- **这解释了为什么职业玩家在湿润牌面上更倾向于激进行动** — 宽范围在这类牌面有更好的胜率实现

### 4. 15% vs 30% (用户关注的对决)

```
翻前:      55.76% vs 44.24%
AK7 干燥:  56.27% vs 43.73%
789 湿润:  50.43% vs 49.57%
```

**15% 的范围打 30% 的范围，翻前有约 56% 的胜率优势。** 这个优势是显著的但不是压倒性的 — 在大样本中会转化为稳定的盈利，但单手牌的波动仍然很大。
