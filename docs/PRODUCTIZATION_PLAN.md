# 红烧扑克工具集产品化计划

目标不是把研究工具变成封闭黑箱，而是把它改造成两层产品:

- **免费研究版**: 开放核心计算、模型解释、单点验证能力，用来建立信任和传播。
- **Pro 工作流版**: 收费卖批量、报告、模板、历史记录和可交付物，让用户少重填、少做表、少解释。

## 一、产品定位

推荐定位:

> 中文扑克玩家的本地化研究工具包: MTT 股份、Day2 估值、赏金跟注、资金升级和手牌复盘。

避免定位:

- 不做牌局平台。
- 不做交易撮合。
- 不处理资金。
- 不组织赛事。
- 不承诺盈利。
- 不引导真实下注。

这能保留研究工具的安全边界，也更适合 H5/PWA、桌面包和后续小程序学习版。

## 二、免费与 Pro 边界

### 免费研究版

免费版负责证明模型可信、工具好用:

| 模块 | 免费能力 |
|---|---|
| 胜率计算器 | 单次范围 vs 范围、手牌 vs 手牌、公共牌输入、基础矩阵 |
| 赏金跟注 | 单 spot PKO / Mystery bounty 所需胜率 |
| MTT 卖股 | 单场买入、人数、ROI、bankroll 的推荐比例 |
| Staking Solver | 单个合约变量反推 |
| Day2 估值 | 单人或少量手动估值 |
| 资金管理 | 单次 bankroll 路径、基础 Monte Carlo |
| 手牌记录器 | 本地记录、复制文本、导出 JSON |
| 文档 | 公式说明、benchmark、使用边界、灵感来源 |

免费版原则:

- 核心公式和单点计算尽量开放。
- 所有计算本地完成。
- 不提供云同步。
- 不提供批量导入导出。
- 不生成专业报告。
- 不保存跨项目工作空间。

### Pro 工作流版

Pro 版不卖“更准的答案”，卖可复用工作流:

| Pro 能力 | 付费理由 |
|---|---|
| 卖股报告生成 | 把计算结果变成可发给投资人/朋友/社群的一页说明 |
| Day2 批量估值 | 一次导入多个玩家筹码，批量输出合理报价和敏感性区间 |
| 赛事模板库 | 常见结构、奖池、PKO/Mystery 参数不用重复录入 |
| 多方案对比 | buy-in、field、ROI、markup、卖出比例并排比较 |
| 本地历史记录 | 保存我的 bankroll、ROI 档案、常用参数和历史报告 |
| Excel / PDF / 长图导出 | 输出可交付物，不只是屏幕上的结果 |
| 私有范围库 | 保存常用 preflop range、翻后命中率场景、训练模板 |
| 教练/社群包 | 可定制模板、批量表、参数校准和讲解 |

建议价格:

- 早鸟版: ¥99-129。
- 正式 Pro: ¥168-199。
- 教练/社群版: ¥399-999。

交付承诺建议写成:

> 买断当前大版本，含 12 个月功能更新。

不要承诺永久无限更新。

## 三、仓库改造目标

### 当前问题

1. 首页按工具类型分组，用户很难第一眼看出“我来这里能完成什么工作”。
2. README 和 footer 写“完全开源”，会限制后续 Pro 包装。
3. 数据资产没有单独位置，赛事结构、奖池模板、范围模板以后会散落到工具里。
4. 免费和 Pro 边界没有文档化，后续开发容易把核心计算和商业功能混在一起。

### 改造方向

```
docs/
  PRODUCTIZATION_PLAN.md   # 本文，产品边界和路线
data/
  tournament-structures/   # 外部赛事结构、奖池、bounty 模板
packages/
  core/                    # 永远保持免费/开源/纯计算
tools/
  ...                      # 免费研究版入口
pro/                       # 未来 Pro 工作流入口和本地存储层
```

核心原则:

- `packages/core/` 保持纯计算、可开源、无 DOM、无授权逻辑。
- `tools/` 是免费研究版。
- `data/` 放可审计的数据模板和来源记录。
- `pro/` 未来只做工作流: 模板、导出、批量、历史、激活码。

## 四、外部需要补充的数据

### P0: 通用模板，可先人工整理

| 数据 | 用途 | 备注 |
|---|---|---|
| 通用 MTT payout 曲线 | ICM、Day2 估值、FT deal | 用 10%-15% ITM 的通用曲线即可先上线 |
| PKO 参数模板 | 赏金跟注、PKO 卖股 | 起始赏金、bounty 占比、递增逻辑 |
| Mystery bounty 模板 | 神秘赏金 EV | 剩余盲盒奖金表、剩余人数、覆盖关系 |
| 常见 field size 档位 | 方差/ROI 模型 | 100、300、500、1000、2000、5000 |
| 常用 bankroll 风格 | 卖股比例解释 | 保守、平衡、激进 |

### P1: 公开赛事资料，需记录来源

| 数据 | 可能来源 | 注意 |
|---|---|---|
| WSOP / WPT / EPT / APT 公开结构表 | 官方 event PDF / 官网 | 只保存自己整理后的参数，不复制整份 PDF |
| 线上平台常见 payout | 平台公开 lobby / help 页面 | 规则会变，标注采集日期 |
| 常见 Day2 结构 | 用户手工输入 / 官方赛程 | 先做导入格式，不急着内置 |
| 中文现场赛 buy-in 和结构 | 官方海报 / 公告 | 避免使用未经授权的宣传图 |

### P2: 个人化数据，不应公开收集

| 数据 | 存储建议 |
|---|---|
| 用户 bankroll、ROI、历史报价 | 本地浏览器 IndexedDB / 桌面本地文件 |
| 投资人/玩家名单 | 本地保存，导出文件由用户自己保管 |
| 私人赛程和报价 | 本地 workspace，不上传 |

## 五、首个 Pro MVP

最小可卖版本只做 3 件事:

1. **卖股报告生成器**
   - 输入当前 MTT 卖股计算结果。
   - 输出一页报告: 参数、推荐卖出比例、markup、投资人视角、风险说明。
   - 支持长图 / PDF / Markdown 导出。

2. **Day2 批量估值**
   - CSV / 表格粘贴多个玩家筹码。
   - 输出每个玩家的零和 ROI、合理 markup、买家 ROI 区间。
   - 支持排序和导出 Excel。

3. **赛事模板库**
   - 通用 MTT、PKO、Mystery、FT deal 几类模板。
   - 每个模板记录来源、更新时间、适用范围。
   - 免费版可看少量模板，Pro 可保存/复制/自定义模板。

## 六、发布路径

### 第一步: H5/PWA Pro

- 继续用现有静态站。
- Pro 页面可先用本地激活码。
- 数据存在 localStorage / IndexedDB。
- 报告导出和批量导入作为付费点。

### 第二步: 桌面离线包

- 用 Tauri 打包同一套前端。
- 卖点是离线、私密、本地文件导入导出。

### 第三步: 小程序学习版

- 只放概率、范围、手牌复盘、资金规划。
- 不放卖股、买股、markup、投资人、交易报告作为主功能。

## 七、合规边界

所有公开文案都应该强调:

- 仅用于概率研究、资金管理模拟和牌局复盘。
- 不提供赌博、下注、资金托管、交易撮合或赛事组织。
- 计算结果依赖用户输入假设，不构成投资建议。

禁用或谨慎使用的公开词:

- 保证盈利、稳赚、下注建议、买入推荐、交易撮合、俱乐部房间、资金托管。

可用词:

- 研究、复盘、概率、模拟、参数、报告、估值、风险说明。
