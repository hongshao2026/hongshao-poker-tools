语法说明 
PokerStove 范围语法:
  AA          —— 一对 A
  AKs         —— 同花 AK
  AKo         —— 不同花 AK
  AK          —— 所有 AK (同花+不同花)
  QQ-TT       —— QQ, JJ, TT
  ATs+        —— ATs, AJs, AQs, AKs
  AJo+        —— AJo, AQo, AKo
  77+         —— 77, 88, 99, ..., AA
  多个范围用逗号分隔: TT+,AQs+,AKo

Board 语法:
  空字符串    —— 翻前
  AhKd2c      —— 翻牌
  AhKd2c5s    —— 转牌
  AhKd2c5sQs  —— 河牌
本地运行说明 
WASM 不能从 file:// 直接加载,需要本地 HTTP 服务器:

  cd /path/to/hongshao-poker-tools
  python3 -m http.server 8000
  # 然后访问 http://localhost:8000/tools/range-zen/

或部署到任何静态托管服务 (Cloudflare Pages, Vercel, Netlify, GitHub Pages)。
