06 / NEGOTIATION

谈判要点

怎么用这两个数
关于技术优势：这个版本不再问"你的 edge 是多少"。原因是：人对自己 edge 的估计普遍偏高（业界研究显示绝大多数玩家高估自己 3–5 个百分点），即使是职业牌手对单场对手的 edge 估计通常也不超过 5–10%。更稳妥的做法是从 ICM 公平价出发谈判，再根据你对自己的信心适当上调开价——确信有显著技术优势，把"接受门槛"定在 ICM 价的 5–10% 之上；否则就以 ICM 公平价为基准。
开价策略：计算出的"公平价"是双方都该认可的中间点，不是你的开价。第一次报价应该比公平价高 5–10%——你永远可以谈下来，但很难谈上去。大多数对手没做过这种计算，他们累、被钱吓到、容易被施压。一种由数学支撑的不可动摇的自信，往往比数学本身更值钱。
